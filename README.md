# E-Commerce-Sales-Analytics-Dashboard
An end-to-end E-Commerce Sales Analytics project using MySQL and Power BI. It involves SQL-based data cleaning and analysis, followed by an interactive dashboard to visualize sales, profit, customer behavior, regional performance, and business trends for data-driven decision-making.
